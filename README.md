# API REST de Produtos

Projeto acadêmico desenvolvido em Java com Spring Boot, com o objetivo de criar
uma API REST para cadastro e gerenciamento de produtos.

## Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Atualização de dados
- Remoção de produtos

## Tecnologias Utilizadas
- Java
- Spring Boot
- Spring Data JPA
- Maven

## Objetivo do Projeto
Aplicar conceitos de desenvolvimento backend, APIs REST e boas práticas
utilizando o framework Spring Boot.
